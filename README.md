# Lesaal Marketing (Next.js)

Modern marketing landing page rebuilt with:

- Next.js (App Router)
- GSAP + ScrollTrigger for large scroll-driven motion
- Lenis for smooth scrolling
- Framer Motion for UI interactions

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Project structure

- `app/` - Next.js app entry and global styles
- `components/` - smooth scroll and animated page components
- `data/` - page content data
- `public/images/services/` - service image assets
